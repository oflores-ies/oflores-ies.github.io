Benvinguts als Projectes

Em dic Ó𝓼𝓬𝓪𝓻♞, sóc alumne de Robòtica de 1r de Batxillerat de l'institut Pompeu Fabra

Aquesta pàgina està escrita en html (llenguatge de marques d'hipertext) que dona l'estructura física i utilitza CSS ( fulles d'estil en cascada) que ens ajuda a donar la forma i l'estil a la web.

A més utilitza JavaScript que proporciona tota la interectivitat de la pàgina web.

Utilitza biblioteca externes com bootstrap que conté JavaScript i CSS i que ens permet fer un menú desplegable i posar un logo fàcilment.

Una altra lliberia que utilitzem és fontawesome, és una biblioteca que ens permet usar símbols amb facilitat, són els símbols que hi són al menú.
